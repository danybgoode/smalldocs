Current link model

Generated reports use hash URLs such as /docs#md=.... Browsers do not send the hash fragment to the server, so the report opens anonymously and the document payload stays client-side.

Deferred registry

Friendly real links like /r/pmo-weekly-2026-07-14 need an explicit report registry, storage backend, write credential, and retention rule. That is a separate decision before this service becomes stateful for PMO reports.
